# 山樱无念

## 简介
山樱无念是基于自然码的音形方案。

优点：
* 重码极低：出一二简和前1500的所有简后，不算多重简码，加权重码仅有0.0096%，出全部简码加权重码仅有0.0045%
* 字词不重：使用特殊的编码方式，让字词不会在同一个码位上
* 没有小字：没有小字，拆分直观，使得打生字时也能快速反应
* 见词就打：特殊的编码方式使得可以做到真正的“见词就打”
