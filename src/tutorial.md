# 教程

## 入门
### 观前提醒
本教程面向已经学会小鹤音形的人
### 简单鹤与小鹤音形的主要区别
1. 简单鹤字根为乱序排列，非音托字根
2. 没有小字
3. 拆分规则不同
4. 简单鹤四码单字统一放在次选。简单鹤采用了极端的字词避重方式，在算码的时候尽量让更多的字处在三码位，剩下的四码单字则全部放在次选，好让四码首选永远可以畅快打词
5. 出简也出全，且不会让位给其他的字

### 简单鹤拆分规则
笔顺优先（无笔画粘连的口可无视笔顺），首末取大，相交也拆，插隔也取，字根字重复，禾取丿八
例如：由 = 日 + 丨， 平 = 一 + 十， 虫 = 虫 + 虫

### 无理码
详见 [无理码](/irregular.md)

### 入门步骤
1. 使用字根练习器练习简单鹤字根
2. 找到简单鹤QQ群并加入，使用QQ群机器人彼方（かなた）获得群号，不会就问群友
3. 打文章练习
