# 白鸠

白鸠是九键音形方案

![ 字根图 ](/dovepeliazgt.png)
1, 2, 3, 4, 5 对应 阴平（ā）、阳平（á）、上声（ǎ）、去声（à）、轻声（a）

## 规则

* 单字：声 介调 韵 形首 形末
* 词语：每个单字的音

## 拼写

1. 把韵母还原为原始的形式：
    * iu -> iou
    * in -> ien
    * ing -> ieng
    * ui -> uei
    * un -> uen
    * vn -> ven
2. 还原v：ju qu xu yu -> jv qv xv yv
3. 轻声合并到一声
4. 只有助词是轻声，其他词不是轻声，如：我的：wo3de5，桌子：zhuo1zi3，“的”是轻声，“子”是上声

## 拆分

顺取首末

带有优先级的拆分规则如下：
1. 根少优先
2. 结构完整
3. 散连交
4. 符合笔顺
5. 字根取大
